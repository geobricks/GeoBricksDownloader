Parameters:
  • source (String) - e.g. 'modis', either lower or upper case.
  • file_system_structure (String | Dict) - This parameter can be either a String, representing the target directory for the downloads (e.g. '/home/user/Desktop'), or a Dict, describing the file system structure (e.g. {'target': '/home/kalimaha/Desktop/MODIS', 'product': 'MOD13Q1', 'year': '2014', 'day': '033'})
  • file_paths_and_sizes (Array) - Collection of objects containing the following fields: 'file_name', 'size', 'file_path', 'label'.
  • threaded (Boolean) - Run the downloader in multiple or single thread mode.
  • username (String) - Optional parameter.
  • password (String) - Optional parameter.
